# دشبورد پیش‌بینی آب و هوا - 5 روز آینده

یک دشبورد زیبا و مدرن برای نمایش پیش‌بینی آب و هوای 5 روز آینده با رابط کاربری فارسی.

## ویژگی‌ها

- 🌤️ **نمایش آب و هوای فعلی** با جزئیات کامل
- 📅 **پیش‌بینی 5 روز آینده** با اطلاعات دقیق
- 🔍 **جستجوی شهر** با قابلیت تایپ و جستجو
- 📍 **موقعیت جغرافیایی** با استفاده از GPS
- 📱 **طراحی ریسپانسیو** برای موبایل و دسکتاپ
- 🎨 **رابط کاربری مدرن** با انیمیشن‌های زیبا
- 🇮🇷 **زبان فارسی** و راست به چپ

## نحوه استفاده

### 1. اجرای فایل‌ها
فایل `index.html` را در مرورگر باز کنید.

### 2. جستجوی شهر
- نام شهر مورد نظر را در کادر جستجو تایپ کنید
- روی دکمه جستجو کلیک کنید یا Enter بزنید

### 3. استفاده از موقعیت فعلی
- روی دکمه "موقعیت فعلی" کلیک کنید
- اجازه دسترسی به موقعیت جغرافیایی را بدهید

## تنظیمات API

برای استفاده از داده‌های واقعی آب و هوا:

1. به [OpenWeatherMap](https://openweathermap.org/api) بروید
2. حساب کاربری ایجاد کنید
3. API Key دریافت کنید
4. در فایل `script.js` خط 4 را ویرایش کنید:
   ```javascript
   this.apiKey = 'YOUR_API_KEY'; // کلید API خود را اینجا قرار دهید
   ```
5. خط 185 را تغییر دهید:
   ```javascript
   const hasApiKey = true; // true کنید
   ```

## سفارشی‌سازی

### تغییر رنگ‌ها
در فایل `styles.css` متغیرهای CSS را تغییر دهید:

```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #ffd700;
}
```

### اضافه کردن شهرهای پیش‌فرض
در فایل `script.js` آرایه `defaultCities` را ویرایش کنید.
