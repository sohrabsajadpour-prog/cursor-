# دشبورد پیش‌بینی آب و هوا - 5 روز آینده

یک دشبورد زیبا و مدرن برای نمایش پیش‌بینی آب و هوای 5 روز آینده به زبان فارسی.

## ویژگی‌ها

- 🌤️ نمایش آب و هوای فعلی
- 📅 پیش‌بینی 5 روز آینده
- 🔍 جستجوی شهر
- 📍 استفاده از موقعیت جغرافیایی
- 📱 طراحی ریسپانسیو
- 🎨 رابط کاربری مدرن و زیبا
- 🌍 پشتیبانی کامل از زبان فارسی

## نحوه استفاده

1. فایل `index.html` را در مرورگر باز کنید
2. برای جستجوی شهر، نام شهر را در کادر جستجو وارد کنید
3. برای استفاده از موقعیت فعلی، روی دکمه موقعیت کلیک کنید
4. اطلاعات آب و هوا و پیش‌بینی 5 روز آینده نمایش داده می‌شود

## اجرای پروژه

### روش 1: اجرای مستقیم
فایل `index.html` را مستقیماً در مرورگر باز کنید.

### روش 2: استفاده از سرور محلی
```bash
# با Python
python -m http.server 8000

# یا با Node.js
npx http-server
```

سپس به آدرس `http://localhost:8000` بروید.

## سفارشی‌سازی

### اضافه کردن API کلید واقعی
برای استفاده از API واقعی OpenWeatherMap:

1. از [OpenWeatherMap](https://openweathermap.org/api) API کلید دریافت کنید
2. در فایل `script.js`، خط 4 را ویرایش کنید:
```javascript
this.apiKey = 'YOUR_ACTUAL_API_KEY';
```

### تغییر شهر پیش‌فرض
در فایل `script.js`، خط 6 را ویرایش کنید:
```javascript
this.currentCity = 'نام شهر مورد نظر';
```
