# داشبورد آب و هوا - پیش‌بینی 5 روزه

یک داشبورد مدرن و زیبا برای نمایش اطلاعات آب و هوا با پیش‌بینی 5 روز آینده.

## ویژگی‌ها

- 🌤️ نمایش وضعیت فعلی آب و هوا
- 📅 پیش‌بینی 5 روز آینده
- 🕐 پیش‌بینی ساعتی (12 ساعت آینده)
- 📍 تشخیص موقعیت مکانی
- 🔍 جستجوی شهرها
- 📱 طراحی ریسپانسیو
- ✨ انیمیشن‌ها و افکت‌های زیبا
- 🌙 رابط کاربری مدرن با پشتیبانی از فارسی

## جزئیات نمایش داده شده

### آب و هوای فعلی
- دمای فعلی و احساس دما
- شرح وضعیت آب و هوا
- رطوبت هوا
- سرعت باد
- میزان دید
- فشار هوا
- شاخص UV

### پیش‌بینی 5 روزه
- نام روز و تاریخ
- دمای حداکثر و حداقل
- آیکون وضعیت آب و هوا
- توضیح مختصر

### پیش‌بینی ساعتی
- دما برای 12 ساعت آینده
- آیکون وضعیت آب و هوا
- زمان دقیق

## نحوه استفاده

1. فایل `index.html` را در مرورگر باز کنید
2. نام شهر مورد نظر را در کادر جستجو وارد کنید
3. روی دکمه جستجو کلیک کنید یا Enter بزنید
4. برای استفاده از موقعیت فعلی، روی دکمه موقعیت کلیک کنید

## میانبرهای صفحه کلید

- `Ctrl/Cmd + K`: فوکوس روی کادر جستجو
- `Ctrl/Cmd + L`: استفاده از موقعیت فعلی
- `Enter`: جستجو در کادر جستجو

## فایل‌ها

- `index.html`: ساختار HTML صفحه
- `styles.css`: استایل‌ها و طراحی UI
- `script.js`: منطق برنامه و تعامل با API

## توجه

این نسخه از داده‌های شبیه‌سازی شده استفاده می‌کند. برای استفاده در محیط واقعی:

1. یک کلید API از OpenWeatherMap دریافت کنید
2. کلید API را در متغیر `apiKey` در فایل `script.js` قرار دهید
3. کامنت‌های مربوط به API واقعی را فعال کنید
